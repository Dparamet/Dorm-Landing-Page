# Image Guide

Store all page images in this folder.

Recommended structure:
- /public/images/dorm/hero-main.jpg
- /public/images/dorm/room-01.jpg
- /public/images/dorm/room-02.jpg
- /public/images/dorm/room-03.jpg

Best practices:
- Use WebP when possible for faster loading.
- Keep hero images under 300KB.
- Use descriptive file names to improve content workflow.
- Add meaningful alt text in data/landing-page-content.js.
